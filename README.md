# Painel do Veículo — v1

App 100% estático, um único arquivo (`index.html`). Sem login, sem Firebase, sem servidor.
Os dados ficam salvos no navegador do próprio aparelho (localStorage).

## Como colocar no ar
1. No repositório do GitHub: **Add file → Upload files** → arrasta o `index.html` → **Commit changes**.
2. **Settings → Pages** → Source: **Deploy from a branch** → Branch **main**, pasta **/ (root)** → **Save**.
3. Espera 1-2 minutos e abre o link (ex: `https://texug037-hue.github.io/manuten-o/`).

## Importante
- Os dados ficam só nesse navegador/aparelho. Se trocar de celular ou limpar os dados do
  site, o histórico se perde. Não tem sincronização entre aparelhos nessa versão.
- Para atualizar o app depois: só substitui o `index.html` no GitHub — o Pages atualiza sozinho.
